Mail Proxy GUI

Welcome to the super simple admin environment for Mail Proxy GUI. Here you can edit the data in the database.

The only thing you need to add to get started are servers. Simply add a new record and fill in the fields.
Please note that the internal IMAP and SMTP hosts need to be without encryption. Port 143 and 25 usually works. Add the IP of Mail Proxy GUI to your servers firewall if it doesn't work.

Using a DNS key that is not mail.? Change the dns_keys option in the options table.

Did a domain move to another server? Change it to the new server in the domains table.
